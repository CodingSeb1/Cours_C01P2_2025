Distributeurs hydrauliques et pneumatiques

Les distributeurs pneumatiques et hydrauliques sont les pré-actionneurs privilégiés des actionneurs pneumatiques. Ils doivent pouvoir faire circuler le fluide de deux manières différentes pour obtenir les deux mouvements du vérin ou les deux sens de rotation d'un moteur. Ils sont caractérisés par :

  • le nombre d'orifices de passage de fluide et le nombre de position ;

  • leur stabilité : monostable ou bistable ;

  • leur dispositif de commande (mécanique, électrique ou pneumatique).

Les distributeurs les plus fréquemment rencontrés sont :

  • 4/2 et 5/2 pour les vérins double effet

  • 3/2 pour les vérins simple effet

Ces distributeurs peuvent être commandés par pilotage :

Un distributeur à rappel par ressort est dit monostable, sinon, il est dit bistable.

D'un point de vue technologique, les distributeurs (pneumatiques, ou hydrauliques), sont basés sur le déplacement d'un tiroir qui laisse passer le fluide vers une sortie ou une autre selon le sens.
