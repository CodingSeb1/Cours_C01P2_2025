Matière d'œuvre et valeur ajoutée

Définition : Matière d'œuvre

La matière d'œuvre (MO) est une entité du milieu extérieur sur laquelle le système agit. Elle peut être de trois types :

  • Matière : matières premières, produits manufacturés, êtres humains, etc.

  • Énergie : énergie électrique, mécanique, hydraulique, pneumatique, thermique, etc.

  • Information : grandeur physique, signal logique (0 ou 1), etc.

Définition : Valeur ajoutée

La valeur ajoutée (VA) est la finalité d'un système : il s'agit de ce qu'apporte le système à la matière d’œuvre entrante (MOE) pour produire la matière d’œuvre sortante (MOS) :

Exemple : Système : Ouvre portail FAAC

Matière d’œuvre : portail

Valeur ajoutée : déplacement physique du portail (ouverture/fermeture)

Energie utilisée : énergie électrique du réseau

Informations entrantes : appui sur la télécommande, présence (ou absence) d'obstacle

Information sortante : signal lumineux du gyrophare
